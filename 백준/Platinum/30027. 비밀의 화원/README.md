### 문제 설명

<p>유진이는 어렸을 적부터 자신만의 화원에서 꽃을 가꾸는 것을 꿈꿔왔다. 많은 세월이 지나고, 꿈을 이루기 위해 열심히 노력한 유진이는 마침내 자신만의 화원을 장만하게 되었다.</p>
